Curso Git: Nuestro primer proyecto.

Comandos Vistos:

  • Comando: git init | Inicializa el proyecto git
  • Comando: git status | Estado del proyecto git
  • Comando: git add archivo.ext | Añade archivos al stagin área
  • Comando: git commit -m "comentatio" | Crea un commit de los archivos modificados.
  • Comando: git log | Lista los commit realizados en el proyecto.
  • Comando: git checkout 'codigo sha commit' | Viajamos en el tiempo en los archivos de cada commit
  • Comando: git checkout master | Volvemos a la rama master al último commit realizado y dejamos todo como estaba antes de viajar en el tiempo
  • Command: git brahch: nos muestra las ramas que existen
  • Command: git brahch Test: crea una nueva rama llamada Test
  • Command: git checkout Test: Cambia a la nueva rama Test

    Al cambiar de rama, los cambios se reflejan en los archivos reales del proyecto
