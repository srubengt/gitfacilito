Curso Git: Nuestro primer proyecto.

Comandos Vistos:

  • Comando: git init | Inicializa el proyecto git
  • Comando: git status | Estado del proyecto git
  • Comando: git add archivo.ext | Añade archivos al stagin área
  • Comando: git commit -m "comentatio" | Crea un commit de los archivos modificados.
  • Comando: git log | Lista los commit realizados en el proyecto.
